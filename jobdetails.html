Senior Frontend Developer

TechFlow Systems San Francisco, CA (Hybrid) Posted 2 days ago

Job Description

We are looking for an experienced Senior Frontend Developer to join our product team. You will be responsible for building high-performance, scalable web applications that power our core platform. The ideal candidate is passionate about user experience, clean code, and modern web technologies.

As a key member of the engineering team, you will collaborate closely with designers, backend developers, and product managers to deliver world-class solutions to our global customer base.

Key Responsibilities

  • Develop new features and maintain existing codebase using React, TypeScript, and Tailwind CSS.
  • Optimize application performance for maximum speed and scalability.
  • Mentor junior developers and conduct code reviews to ensure high code quality.
  • Collaborate with UX/UI designers to translate designs into pixel-perfect code.

Requirements

  • 5+ years of experience in frontend development.
  • Proficiency in JavaScript (ES6+), TypeScript, and React ecosystem.
  • Experience with state management libraries (Redux, Zustand, or Context API).
  • Strong understanding of RESTful APIs and modern build tools (Vite, Webpack).
